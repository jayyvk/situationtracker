# Situation Tracker

Dashboard to monitor the current geopolitical situation in the middle east.

![IMG_5167](https://github.com/user-attachments/assets/69736552-0ccb-4d4d-8a76-fc08cf68302b)

It combines:
- a live news feed
- a situational map with hotspot markers
- embedded live news video
- market widgets
- a curated resources panel

## Notes

This project is still evolving. Some data sources are live, while others are currently simplified or embedded for v1.
